# West Peek Brand Guide — Founder Dilution Dashboard

## Product name

Founder Dilution Dashboard by West Peek

## Logo

Use the WP mark from `public/brand/wp-logo.png` in the app header and branded surfaces.

Alt text: `West Peek logo`

## Palette

| Token | Use |
|---|---|
| West Peek Orange `#f05a1a` | Primary accent, active nav, key warnings, major numbers, buttons |
| Orange Dark `#b83c08` | Accessible orange text on light backgrounds |
| Black `#050505` | Primary text, headers, serious emphasis |
| White `#ffffff` | Cards and product surfaces |
| Warm Cream `#fbfaf7` | App background |
| Border `#e6ded3` | Card borders and quiet dividers |

## Usage rules

- Orange is a signal, not wallpaper. Use it for attention and action.
- Black carries authority. Use it for headings, navigation, tables, and decisive labels.
- White keeps the product calm and founder-friendly.
- Warning states must include text, not color alone.
- The product must continue to read like a founder guide, not an internal finance or engineering tool.

## User-facing language

Use plain-English founder language:

- Payment order
- Investor payback rights
- Investor gets paid twice
- Money left after costs
- Founder/team shares
- Sale price needed before you make money

Avoid internal or technical UI language.
